Database and utilities for body, outfit and preset mods.

Fetches mod information and stores them in a local database using LiteDB.  
Uses the Nexus Mods API v1 and v2 (v2 being the GraphQL API).

Can also find, download and extract BodySlide presets.

For downloading presets, adding a user secret "nexusMods:apiKey" is required.  
You can create an API key under the settings of your Nexus Mods account.
